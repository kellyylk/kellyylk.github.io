    • AOIFE

    (2018)

    Aoife is an object I created using and experimenting with Arduino. This object is an interactive home decoration that rotates 360 degrees when you’re 150cm close to it. I got inspiration from modern architecture, the way they possess sleek, geometric forms. Many home decorations I usually see tend to be stationary, when I learned about what certain components in a microcontroller kit can do, I saw a lot of opportunities to bring home decoration into life. I began drawing geometric shapes spontaneously, having them overlay on top of each other. Then, I connected some of the fragments together, creating a new shape and have them cut.
